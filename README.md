### 英语自主学习系统+Vue3++SpringBoot
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL、VSCode
技术栈：Java、MySQL、SpringBoot、Mybatis/MybatisPlus、Vue3、ElementUI、HTML、JavaScript、CSS

系统功能模块：
本系统包含管理员、用户2个角色。
管理员角色：
1. 用户管理：管理系统中所有用户的信息，包括添加、删除和修改用户。
2. 配置管理：管理系统配置参数，如轮播图等。
3. 试卷管理：管理试卷的创建、修改和删除。
4. 试题库管理：管理试题的添加、修改和删除。
5. 单词库管理：管理单词的添加、修改和删除。
6. 教学视频管理：管理教学视频的添加、修改和删除。
7. 论坛管理：管理论坛帖子的查看和删除。
8. 每日一句管理：管理每日一句的添加、修改和删除。
用户角色：
1. 单词学习：用户可以查看和学习单词库中的单词。
2. 试卷列表：用户可以查看试卷列表并进行考试。
3. 论坛交流：用户可以发布和查看论坛帖子。
4. 教学视频：用户可以查看教学视频。
5. 考试记录：用户可以查看自己的考试记录。
6. 我的收藏：用户可以查看和管理自己的收藏内容。
